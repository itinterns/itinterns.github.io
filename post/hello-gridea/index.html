👏 Welcome to Gridea!
** Gridea A static blog writing client. You can use it to record your life, mood, knowledge, notes, ideas...

[Github] (https://github.com/getgridea/gridea)
[Gridea Home] (http://hvenotes.fehey.com/)
[example website] (http://fehey.com/)

Characteristic👇

📝 You can use the coolest Markdown syntax for quick authoring

🌉 You can attach a beautiful cover image to the article and insert a picture anywhere in the article.

🏷️ You can group tags into articles

📋 You can customize the menu and even create an external link menu

💻 You can use this client on Windows or MacOS devices

🌎 You can use Github Pages or Coding Pages to show the world that more platforms will be supported in the future.

💬 You can perform simple configuration and access [Gitalk] (https://github.com/gitalk/gitalk) or [DisqusJS] (https://github.com/SukkaW/DisqusJS) comment system

🇬🇧 You can use Chinese Simplified or English

🌁 You can use the in-app default theme or any third-party theme at will, powerful theme customization capabilities.

🖥 You can customize the source folder to synchronize multiple devices with OneDrive, Baidu network disk, iCloud, Dropbox, etc.

🌱 Of course Gridea is still very young, there are many deficiencies, but please believe that it will keep moving forward

In the future, it will definitely be a partner you can't live without.

Make the most of your talents!

😘 Enjoy~
